YT2MPV

Simple installer that adds sponsorblock, subtitles, quality switching and URI for watching youtube in MPV. Currently pulls cookies and marks videos as watched [Only for Firefox]. For best results use a redirector plugin to change urls from https://www.youtube.com/watch?v=videoid to yt://www.youtube.com/watch?v=videoid

INSTALL:
unzip and sh install.sh
